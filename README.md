# coreclr-udpsend

A simple UDP datagram send client and echo back server by using CoreCLR.

Tested on `CoreCLR 1.0.0-preview2-003131`

## Run as server

```bash
dotnet run localhost 61000
```

## Run as client

```bash
dotnet run <serverHostname> 61000
```
